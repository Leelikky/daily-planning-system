# Daily Planning System

> **让 AI 真正「懂你」的规划与记忆系统**
>
> 一个基于五层记忆架构的闭环工作流，帮你清空大脑、对齐目标、沉淀自我认知。

---

## 为什么你需要这个系统？

你是否遇到过这些问题：

- 🧠 **大脑过载**：想做的事太多，不知道从哪里开始
- 🎯 **目标漂移**：忙碌一天，却发现偏离了真正重要的事
- 📝 **决策反复**：每次都要重新解释你的偏好和风格
- 💭 **自我迷失**：不清楚自己的行为模式和价值观

**Daily Planning System** 通过「记忆 + 规划 + 复盘」的闭环，让 AI 越用越懂你，帮你从混乱中建立秩序，从重复中发现规律。

---

## 核心理念：一个闭环，三层价值

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│     ┌──────────────┐      ┌──────────────┐      ┌───────────┐  │
│     │   记忆系统    │ ──▶ │   每日规划    │ ──▶ │  执行行动  │  │
│     │  (你是谁)    │      │ (今天做什么)  │      │ (做了什么) │  │
│     └──────────────┘      └──────────────┘      └───────────┘  │
│            ▲                                           │        │
│            │              ┌──────────────┐              │        │
│            └──────────────│   每日复盘    │◀─────────────┘        │
│                           │ (沉淀了什么)  │                       │
│                           └──────────────┘                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

| 环节 | 作用 | 产出 |
|------|------|------|
| **早间规划** | 参考记忆系统，对今日任务进行价值排序 | 今日执行清单 |
| **互动确认** | 与 AI 对话，明确当天工作重心 | 清晰的优先级 |
| **晚间复盘** | 总结行为，将新洞察沉淀回记忆系统 | 更懂你的 AI |

## 一、记忆系统：五层架构，从对话到核心认知

记忆系统采用「被动沉淀」轨道，让 AI 通过日常对话逐步了解你：

```
┌─────────────────────────────────────────────────────────────┐
│                        L4 核心层                              │
│              核心价值观 · 人生信念 · 决策原则                   │
│                    ⚠️ 仅允许人工修改                           │
├─────────────────────────────────────────────────────────────┤
│                        L3 认知层                              │
│              思维模式 · 决策框架 · 心智模型                     │
│                   年度复盘确认后沉淀                           │
├─────────────────────────────────────────────────────────────┤
│                        L2 行为层                              │
│              工作习惯 · 沟通风格 · 时间模式                     │
│                   季度复盘确认后沉淀                           │
├─────────────────────────────────────────────────────────────┤
│                        L1 情境层                              │
│              近期事件 · 项目状态 · 待办事项                     │
│                   保留 30 天，自动清理                          │
├─────────────────────────────────────────────────────────────┤
│                        L0 状态层                              │
│                   当前对话的临时状态                            │
│                     会话结束即消失                              │
└─────────────────────────────────────────────────────────────┘
```

### 沉淀规则

| 层级跃迁 | 触发条件 |
|----------|----------|
| L0 → L1 | 会话结束、重要事件发生 |
| L1 → L2 | 同一行为模式出现 3 次以上 |
| L2 → L3 | 发现稳定的思维模式 |
| L3 → L4 | 年度深度反思（人工确认） |

### 主动输入轨道（Intent）

除了被动沉淀，你也可以主动告诉 AI：

| 文件 | 用途 |
|------|------|
| `目标与规划.yaml` | 短期/中期/长期目标 |
| `偏好与要求.yaml` | 交互风格、内容格式偏好 |
| `约束与边界.yaml` | 决策红线、禁忌事项 |

## 二、Daily Plan Skill：规划与复盘的完整闭环

### 🌅 早间：今日规划

```
你说：「开始今日规划」
```

**Skill 会做什么：**

1. **加载记忆** → 读取你的目标、偏好、价值观
2. **脑倒空** → 让你倾倒所有想做的事，不评判
3. **价值排序** → 基于你的核心层，计算每个任务的优先级
4. **任务拆分** → 区分「保护时间任务」和「Agent 自动任务」
5. **生成清单** → 输出今日执行表，自动保存

**优先级公式：**
```
优先级分数 = 目标匹配度(40%) + 重要性(30%) + 紧急性(20%) + 精力匹配(10%)
```

**时间块匹配：**
- 早上 → 高精力任务（复杂思考、创意工作）
- 下午 → 中等精力（沟通协作、学习）
- 晚上 → 低精力（整理、探索）

### 🌙 晚间：今日复盘

```
你说：「开始今日复盘」
```

**Skill 会做什么：**

1. **读取规划** → 对照早上的计划
2. **引导提问** → 5 个问题帮你梳理一天
3. **生成报告** → 完成情况、关键收获、待沉淀洞察
4. **更新记忆** → 将新洞察写入 L1 情境层
5. **沉淀候选** → 发现阶段行为模式，加入沉淀队列

**复盘问题清单：**
1. 今天实际完成了哪些任务？
2. 有哪些任务没完成？原因是什么？
3. 今天有什么意外的收获或发现？
4. 今天的精力状态如何？
5. 明天有什么重要事项？

## 三、快速开始

### 1. 安装 Skill

```bash
# 将 skill 目录复制到 Claude Code 的 skills 目录
cp -r skill/daily-plan ~/.claude/skills/   # macOS/Linux
cp -r skill/daily-plan "C:/Users/[用户名]/.claude/skills/"   # Windows
```

### 2. 修改路径

编辑 `skill/daily-plan/SKILL.md`，将所有 `[你的项目路径]` 替换为实际路径。

### 3. 初始化记忆系统

打开以下文件，填写你的初始信息：

- `memory-system/Memory/L4_核心层.yaml` → 你的核心价值观、决策原则
- `memory-system/Intent/目标与规划.yaml` → 你的短期/中期/长期目标

### 4. 开始使用

```
开始今日规划    →  进入脑倒空与优先级排序
开始今日复盘    →  总结一天并沉淀记忆
```

## 六、设计初衷

> 我希望建立一套与 AI 交互的记忆系统，帮助我抽象和提炼出自己的价值观、性格特征、心智模式、思维模式、工作习惯、行为偏好、决策原则。让 AI 越用越懂我，甚至能模拟我的决策。

这个系统的核心价值：

- **让 AI 真正懂你**：不是每次都要重新解释
- **让目标清晰可见**：每次规划都对照长期目标
- **让成长有迹可循**：沉淀下来的就是你的成长轨迹
